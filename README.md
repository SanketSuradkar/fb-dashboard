# 📊 Facebook Dashboard

A full-stack web application that allows users to log in with Facebook, view their pages, and analyze key metrics like followers, engagement, and reactions through a clean dashboard interface.

---

## 🚀 Live Demo

👉 https://fb-dashboard-five.vercel.app/

---

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS
* Facebook JavaScript SDK

### Backend

* Node.js
* Express.js
* Axios

---

## ✨ Features

* 🔐 Facebook Login Integration
* 📄 Fetch User Pages
* 📊 Dashboard with:

  * Total Followers
  * Total Fans
  * Total Reactions
  * Total Engagement
  * Estimated Impressions
* 📅 Date Filter (Since / Until)
* 🎨 Clean & Responsive UI

## ⚙️ Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/SanketSuradkar/fb-dashboard.git
cd fb-dashboard
```

---

### 2. Backend Setup

```bash
cd backend
npm install
node index.js
```

---

### 3. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🌐 Deployment

* **Frontend:** Vercel
* **Backend:** Render

---

## 🔑 Facebook App Setup

* Add your domain to **App Domains**
* Enable **Facebook Login**
* Add valid OAuth redirect URIs
* Enable permissions:

  * pages_show_list
  * pages_read_engagement
